# Python TicTacToe

**Installation**  

This project requires Python 2.7.10  

**Install this project by cloning:**  
```
git clone https://github.com/shibani/python-tictactoe.git
```
___

**To run tests**

cd into root of app directory and run in console.

To run the entire suite:

```
pytest 
```
